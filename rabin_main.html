Panduan Algoritma Akar Kuadrat Modulo Prima
Program secara otomatis memilih algoritma yang paling efisien berdasarkan sifat kongruensi modulo 4 dan modulo 8 dari bilangan prima p dan q.
Alg 6.5 p ≡ 3 (mod 4)  →  Metode Langsung
Syarat: p mod 4 == 3
Langkah 1: Hitung r = c^((p+1)/4) mod p
Langkah 2: Kembalikan akar (r, −r mod p)

Contoh: p = 331 → 331 mod 4 = 3.
r = 62111^83 mod 331 = 144. Akar: (144, 187).
Alg 6.6 p ≡ 5 (mod 8)  →  Kasus Khusus
Syarat: p mod 8 == 5
Langkah 1: Hitung d = c^((p−1)/4) mod p
Langkah 2a: Jika d = 1 → r = c^((p+3)/8) mod p
Langkah 2b: Jika d = p−1 → r = 2c·(4c)^((p−5)/8) mod p
Langkah 3: Kembalikan akar (r, −r mod p)
Alg 6.3 Kasus Umum (Fallback)
Syarat: Tidak memenuhi kondisi di atas (p ≡ 1 mod 8). Menggunakan polinomial.
Langkah 1: Cari b acak hingga (b²−4c)/p = −1 (Quadratic Non-Residue).
Langkah 2: Definisikan polinomial f(x) = x²−bx+c.
Langkah 3: Hitung r = x^((p+1)/2) mod f(x).
Langkah 4: Kembalikan akar (r, −r mod p).
Teorema Sisa Tiongkok (CRT)
Menggabungkan akar mod p (r, -r) dan akar mod q (s, -s) menjadi mod n:
1. Gunakan Extended Euclidean untuk mencari koefisien c, d sehingga c·p + d·q = 1.
2. Akar 1: x = (r·d·q + s·c·p) mod n
3. Akar 2: y = (r·d·q − s·c·p) mod n
4. Empat akar yang dihasilkan: x, −x mod n, y, −y mod n.
